# Perceptron Implementation from Scratch

এই প্রোজেক্টে আমি পাইথন এবং নামপাই (NumPy) ব্যবহার করে একদম স্ক্র্যাচ থেকে একটি **Perceptron** নিউরাল নেটওয়ার্ক তৈরি করেছি। এখানে মূলত দেখানো হয়েছে কীভাবে একটি পারসেপট্রন লজিক গেট (AND, OR) সমাধান করতে পারে এবং কেন এটি XOR গেট সমাধানে ব্যর্থ হয়।

## Key Features
- **Custom Perceptron Class:** লার্নিং রেট এবং ইপক কন্ট্রোল করার সুবিধা।
- **Weight Update Logic:** প্রতিটি ভুলের জন্য কীভাবে ওয়েট আপডেট হয় তার বিস্তারিত ইমপ্লিমেন্টেশন।
- **Logic Gate Solving:** AND এবং OR গেটের সফল ট্রেইনিং এবং প্রেডিকশন।
- **XOR Analysis:** কেন পারসেপট্রন নন-লিনিয়ার সমস্যা সমাধান করতে পারে না তার প্র্যাকটিক্যাল ডেমো।



## Technologies Used
- **Python**
- **NumPy** (Mathematical operations)
- **Pandas** (Data manipulation)

## What I Learned
- পারসেপট্রন কীভাবে **Linear Decision Boundary** তৈরি করে।
- **Bias** এবং **Learning Rate** এর গুরুত্ব।
- **Linearly Separable** এবং **Non-linearly Separable** ডেটার মধ্যে পার্থক্য।
